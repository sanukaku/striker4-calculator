# Garmin STRIKER4 照射範囲計算機

Garmin STRIKER4魚群探知機のエコー照射範囲を計算するWebアプリケーション。湖面に浮かべた魚群探知機から湖底への照射範囲を可視化します。

## 公開URL

https://sanukaku.github.io/striker4-calculator/

スマートフォンでアクセスして使用できます。

## 機能

- **3段階のレンジ切り替え**:
  - 水深10mまで: 1m〜10m（0.5m刻み）
  - 水深20mまで: 1m〜20m（1m刻み）
  - 水深50mまで: 1m〜50m（2m刻み）

- **水深選択**: 垂直スライダーで直感的に操作
  - 上が浅く、下が深い自然な配置
  - マウスホイール対応（PC）
  - 水をイメージした青のグラデーション背景
  - デフォルト値: 5m（詳細モード）

- **同時計算表示**:
  - 77Hz（45度）広角照射
  - 200Hz（15度）狭角照射
  - 照射範囲円の直径を小数点第1位まで表示

- **スマホファースト設計**:
  - タッチ操作に最適化
  - 1画面に収まるコンパクトレイアウト
  - スクロール不要
  - レスポンシブデザイン
